    • - retorna uma string contendo o caracter de posição *importante
  • concat(string, string...)
    • - este método retorna uma string contendo string + string (concatena)
  • replace
    • - busca na string uma substring que seja igual ao conteúdo

    • - retorna a substring contendo os caracteres de inicio até o fim
  • split(separator, limit)
    • - retorna um obj. array contendo as substring entre o separator e limit

    • - retorna o resultado da conversão do objeto que chamou o método (inicio, fim)
